# equipauto_scraper_fast.py

Scraper optimisé pour extraire la liste des exposants d'EQUIP AUTO Paris.

**Fonctionnalités**: Acceptation cookies automatique, passage mode carte→liste, parsing BeautifulSoup ultra-rapide.

**Usage**:
```bash
python3 equipauto_scraper_fast.py
```

**Résultat**: 2514 éléments extraits en ~25 secondes → JSON/CSV/Excel dans `output/`
